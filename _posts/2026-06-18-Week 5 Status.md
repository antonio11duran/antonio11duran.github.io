This post is a part of a series of weekly status updates for an Android app project I am doing as a part of the Professional Development in Computer Science class at the University of Colorado Boulder.

## What did you do last week?

Last week I started testing my API call through my app in Android Studio. I was able to reach my goal for the week of calling the WaniKani API and logging it's response. I also wasn't happy with the state of my website, so I dedicated some time to improve and rearrange some things. It still needs work, but it is in a better spot than before.

## What do you plan to do this week?

This week my goal is to take my API call that pulls all reviews and use it to only pull 1 review at a time.

## Are there any impediments in your way?

The biggest impediment is to focus on these smaller tasks first and not try to build the full review queue yet. I keep thinking ahead a few steps before actually implementing the smaller task I have set out to complete. I keep thinking on future problems instead of just finishing this first one!

## Reflection on the process you used last week, how can you make the process work better?

Like I stated a bit above, I keep focusing on the larger structure of the project and worrying how I will get to the finished product. Instead, I need to keep realigning my focus into the small things that need to be implemented. Instead of focusing on my review queue and how I will go from one to the next, I need to just focus on my function to pull a single review. I need the reminder that everything I do is a building block towards the larger structure of the project.
